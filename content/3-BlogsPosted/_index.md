###  [Blog 1 - LOAD TESTING ON AWS](3.1-Blog1/)
This blog introduces Distributed Load Testing on AWS, a solution that helps simulate large traffic and evaluate system performance before going into production. It explains how to create test scenarios, run load testing using containers on AWS Fargate, and monitor results through CloudWatch. This is an important topic for ensuring system stability, scalability, and performance under high traffic conditions.

###  [Blog 2 - AWS WAF: WEB APPLICATION FIREWALL](3.2-Blog2/)
This blog explains AWS WAF and how it protects web applications from common attacks such as SQL Injection, XSS, and DDoS. It highlights how AWS WAF can be integrated with services like CloudFront, ALB, and AWS Shield to build a multi-layer security architecture. This is a key service for improving system security and protecting user data in real-world applications.

###  [Blog 3 - AWS CODEPIPELINE: AUTOMATE YOUR APPLICATION DEPLOYMENT PROCESS](3.3-Blog3/)
This blog introduces AWS CodePipeline, a CI/CD service that automates the build, test, and deployment process. It shows how CodePipeline integrates with other AWS services like CodeBuild, CodeDeploy, S3, and EC2 to create a complete deployment workflow. This helps reduce manual work, minimize errors, and accelerate the software development lifecycle.
